Morrison Hair Co. HTML Website

How to open:
1. Unzip this folder.
2. Double-click index.html.
3. The website opens in your browser. No npm or Node.js needed.

Key links already added:
- Booking: https://www.vagaro.com/leveltennsalon/book-now
- Instagram: https://www.instagram.com/createwithchristina/

How to add photos:
1. Put image files inside the images folder.
2. In index.html, replace a placeholder div with an image tag such as:
   <img src="images/photo-name.jpg" alt="Hair transformation by Christina">

Pricing is currently stored directly inside index.html in the Services section.
